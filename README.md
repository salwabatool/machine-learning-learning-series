# Machine Learning Learning Series 🤖

A complete machine learning learning series covering topics from beginner to advanced, including theory, practical notebooks, and mini-projects.

## 🎯 Goal
Build a GitHub-ready machine learning portfolio that shows progression from fundamentals to advanced topics with clean structure, practical notebooks, mini-projects, and interview/CV value.

## 📚 What This Repository Covers

| # | Module | Topics |
|---|--------|--------|
| 00 | Python for ML | Python basics, NumPy, Pandas, Visualization |
| 01 | Math for ML | Linear algebra, Statistics, Calculus, Optimization |
| 02 | Data Preprocessing | Cleaning, Feature engineering, Encoding, Splitting |
| 03 | Supervised Learning | Regression, Classification, SVM, XGBoost |
| 04 | Model Evaluation | Metrics, Cross-validation, Bias-variance, Tuning |
| 05 | Unsupervised Learning | K-Means, PCA, DBSCAN, Anomaly detection |
| 06 | Feature Selection | Filter, Wrapper, Embedded methods |
| 07 | Ensemble Learning | Bagging, Boosting, Stacking |
| 08 | Deep Learning | Neural networks, CNN, RNN, Transformers |
| 09 | NLP | Text preprocessing, TF-IDF, Embeddings, LLMs |
| 10 | Computer Vision | Image basics, CNN, Transfer learning |
| 11 | Time Series | ARIMA, Prophet, LSTM forecasting |
| 12 | Recommendation Systems | Collaborative & content-based filtering |
| 13 | MLOps & Deployment | FastAPI, Docker, MLflow, Monitoring |
| 14 | Ethics & Production ML | Fairness, SHAP, LIME, Checklists |
| 15 | Projects | 5 end-to-end real-world projects |

## 🚀 Projects Included

1. 🏠 **House Price Prediction** — Regression with feature engineering
2. 📉 **Customer Churn Prediction** — Classification with imbalanced data
3. 💬 **Sentiment Analysis** — NLP text classification
4. 🖼️ **Image Classification** — CNN with transfer learning
5. 🎯 **Recommendation System** — Collaborative filtering mini-project
