# Recommendation System Mini Project

**Goal**: Build a movie recommendation engine.

## Approaches
1. **Collaborative Filtering**: User-based CF with cosine similarity
2. **Content-Based Filtering**: TF-IDF on genres
3. **Hybrid**: Weighted combination of both

## Key Learnings
- Build user-item matrix from ratings
- Compute cosine similarity between users/items
- Handle cold start with content-based fallback
