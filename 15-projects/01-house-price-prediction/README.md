# House Price Prediction

**Goal**: Predict house prices using the California Housing dataset.

## Key Techniques
- Feature engineering (rooms per household, bedroom ratio, log income)
- Regression models: Linear, Ridge, Random Forest, Gradient Boosting
- StandardScaler for preprocessing

## Results
| Model | RMSE | R² |
|-------|------|----|
| Linear Regression | ~0.73 | ~0.60 |
| Ridge | ~0.73 | ~0.60 |
| Random Forest | ~0.50 | ~0.80 |
| Gradient Boosting | ~0.46 | ~0.83 |

## Dataset
California Housing dataset from sklearn
